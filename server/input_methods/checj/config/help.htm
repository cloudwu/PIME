酷倉輸入法模組使用說明：

「`」鍵選單：

你可以把此設定視窗裡的設定視為輸入法的預設設定，
而「`」鍵選單裡的設定則可視為在輸入法臨時切換中、英文輸入...等操作。

與一般輸入法不同的是，PIME 會為每個程式建立一個 CheCJTextService，
所以每個程式下可獨立使用或保持當前的輸入法狀態或設定。

此種方式的好處，最典型的兩個例子：

A 程式為 MPC PLAYER，須切到英文模式，熱鍵才會生效。
B 程式為 記事本，須切到中文模式，才能輸入中文。

當在 B 程式編輯到一半，想切到 A 程式以熱鍵操作 MPC，此時就完全不用再將輸入法切成英文模式。
操作完 A 程式後，回到 B 程式繼續編輯，此時也不用再將輸入法切成中文模式。

A 程式為 記事本，須使用簡體中文編輯資料。
B 程式為 記事本，須使用繁體中文編輯資料。
C 程式為 記事本，尚未開啟。

當在 B 程式編輯到一半，想切到 A 程式編輯資料時，此時就完全不用再將輸入法切成簡中模式。
編輯完 A 程式後，回到 B 程式繼續編輯，此時也不用再將輸入法切成繁中模式。
如果這時突然開 C 程式輸入其它資料，此時輸入法的設定即為在設定視窗的預設設定。

這就是「`」鍵選單功能為何僅定位為臨時變更狀態而不即時儲存的原因。

因為應該沒有人會想為了臨時的操作，而一直在切換各個程式時去重新設定輸入法。
